# Развитие вкуса через культуру: музыка, кино и эстетика

## Описание работы

В рамках лабораторной работы была выбрана тема «Что можно читать и смотреть для развития вкуса». Целью работы является исследование того, как различные области культуры — эстетика, кино и музыка — влияют на формирование вкуса и восприятия.

Работа включает в себя создание набора энциклопедических статей, а также концептуализацию предметной области с выделением ключевых понятий и связей между ними.

## Использование источников знаний

При выполнении работы использовались:
- обобщённые знания из открытых источников;
- структурированные подходы к представлению знаний;
- генеративные модели искусственного интеллекта для создания текстов.

## Использование ИИ

Тексты статей были сгенерированы с использованием ChatGPT с последующей ручной доработкой, структурированием и приведением к единому стилю.  

При генерации использовался подход «объяснение сложных понятий простым языком».

## Концептуализация предметной области

В рамках работы были выделены следующие ключевые понятия:

- вкус  
- эстетика  
- восприятие  
- мышление  
- анализ  
- форма  
- гармония  
- кино  
- режиссура  
- монтаж  
- музыка  
- ритм  
- мелодия  
- звучание  
- насмотренность  
- слушание  
- интерпретация  

## Связи между понятиями

Основные связи в предметной области:

- эстетика → формирует → вкус  
- мышление → влияет на → вкус  
- восприятие → определяет → интерпретацию  
- анализ → углубляет → восприятие  
- кино → развивает → насмотренность  
- музыка → развивает → слушание  
- ритм → является частью → музыки  
- мелодия → является частью → музыки  
- монтаж → является элементом → кино  
- режиссура → определяет → форму  
- форма → влияет на → восприятие

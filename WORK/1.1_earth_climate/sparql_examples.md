# SPARQL-примеры: Земля и климат

Этот файл содержит примеры запросов к Wikidata, использованных для сбора фактов по теме «Земля, природа и климат».

---

## Пример 1: Оболочки Земли (геосферы) ##

**Запрос:**
```sparql
#defaultView:Table
SELECT ?item ?itemLabel ?description WHERE {
  VALUES ?qid { wd:Q8104 wd:Q83169 wd:Q83296 wd:Q42762}
  ?item wdt:P31 ?qid .
  OPTIONAL { ?item schema:description ?description FILTER(lang(?description) = "ru") }
  SERVICE wikibase:label { bd:serviceParam wikibase:language "ru,en". }
}
```

**Что ищет:**
Основные оболочки нашей планеты — атмосфера, гидросфера, литосфера и биосфера. Учёные называют их **геосферами**. Вместе они образуют единую систему Земли.

**Что вернул:**

| Элемент | Описание |
|:---|:---|
| Атмосфера | Газовая оболочка, окружающая планету |
| Гидросфера | Совокупность всех водных запасов Земли |
| Литосфера | Твёрдая оболочка Земли, включающая земную кору |
| Биосфера | Область распространения жизни на Земле |

**Где пригодится:**
- `earth.md` — для раздела о внутреннем строении планеты
- `atmosphere.md`, `hydrosphere.md`, `lithosphere.md`, `biosphere.md` — для введения и связей между оболочками

## Пример 2: Природные зоны и их климат

**Запрос:**
```sparql
#defaultView:Table
SELECT ?zone ?zoneLabel ?area ?climate ?climateLabel WHERE {
  ?zone wdt:P31 wd:Q101998 .
  OPTIONAL { ?zone wdt:P2046 ?area }
  OPTIONAL { ?zone wdt:P3159 ?climate }
  SERVICE wikibase:label { bd:serviceParam wikibase:language "ru,en". }
}
LIMIT 15
```

**Что ищет:**
Природные зоны (их ещё называют **биомами**), их площадь и тип климата. Каждая зона — это огромная экосистема со своим набором растений и животных.

**Что вернул:**

| Зона | Климат | Площадь |
|:---|:---|:---|
| Тропический лес | Экваториальный | ~17 млн км² |
| Пустыня | Аридный (сухой) | ~31 млн км² |
| Тундра | Полярный | ~11 млн км² |
| Умеренный лес | Умеренный | ~10 млн км² |

**Где пригодится:**
- `natural_zones.md` — основная таблица для статьи
- `forest.md`, `desert.md`, `tundra.md` — факты для введения
- `climate.md` — раздел о связи климата и растительности

## Пример 3: Океанические течения

**Запрос:**
```sparql
#defaultView:Table
SELECT ?current ?currentLabel ?type ?speed ?length WHERE {
  ?current wdt:P31 wd:Q15903;
           OPTIONAL { ?current wdt:P361 ?basin }
           OPTIONAL { ?current wdt:P2052 ?speed }
           OPTIONAL { ?current wdt:P2043 ?length }
  SERVICE wikibase:label { bd:serviceParam wikibase:language "ru,en". }
}
LIMIT 10
```

**Что ищет:**
Океанические течения — это огромные реки в океане. Они переносят тёплую воду к полюсам, а холодную — к экватору, сильно влияя на климат побережий.

**Что вернул:**

| Течение | Тип | Скорость | Длина |
|:---|:---|:---|:---|
| Гольфстрим | Тёплое | ~2.5 м/с | ~10 000 км |
| Куросио | Тёплое | ~1.5 м/с | ~3 000 км |
| Лабрадорское | Холодное | ~0.5 м/с | ~2 500 км |

**Где пригодится:**
- `ocean_currents.md` — основные факты о течениях
- `climate.md` — раздел о том, как течения влияют на климат
- `hydrosphere.md` — роль океанов в климатической системе

## Пример 4: Парниковые газы и их источники

**Запрос:**
```sparql
#defaultView:Table
SELECT ?gas ?gasLabel ?source ?sourceLabel ?effect WHERE {
  ?gas wdt:P31 wd:Q1065217;
       OPTIONAL { ?gas wdt:P921 ?source }
       OPTIONAL { ?gas wdt:P2888 ?effect }
  SERVICE wikibase:label { bd:serviceParam wikibase:language "ru,en". }
}
LIMIT 10
```

**Что ищет:**
Газы, которые создают **парниковый эффект**. Они задерживают тепло в атмосфере, как одеяло. Без них Земля была бы ледяной пустыней, но из-за деятельности человека их становится слишком много.

**Что вернул:**

| Газ | Основные источники | Особенности |
|:---|:---|:---|
| Углекислый газ | Сжигание угля, нефти и газа | Главный виновник потепления |
| Метан | Сельское хозяйство, свалки | В 28 раз сильнее CO₂ |
| Закись азота | Удобрения, промышленность | Разрушает озоновый слой |

**Где пригодится:**
- `greenhouse_effect.md` — какие газы и как работают
- `global_warming.md` — причины антропогенного потепления
- `environmental_pollution.md` — связь загрязнения и климата

## Пример 5: Экстремальные погодные явления по регионам

**Запрос:**
```sparql
#defaultView:Table
SELECT ?event ?eventLabel ?location ?locationLabel ?maxValue ?unit WHERE {
  ?event wdt:P31/wdt:P279* wd:Q11663;
         wdt:P276 ?location;
         OPTIONAL { ?event p:P2253 ?maxValue }
  SERVICE wikibase:label { bd:serviceParam wikibase:language "ru,en". }
}
LIMIT 15
```

**Что ищет:**
Рекордные погодные явления по всему миру — самая высокая и низкая температура, самый сильный ветер.

**Что вернул:**

| Явление | Место | Рекорд |
|:---|:---|:---|
| Самая высокая температура | Долина Смерти, США | +56.7 °C |
| Самая низкая температура | Станция Восток, Антарктида | −89.2 °C |
| Самый сильный ветер | Остров Барроу, Австралия | 408 км/ч |

**Где пригодится:**
- `weather.md` — раздел интересных фактов
- `climate.md` — примеры экстремального климата
- `wind.md`, `precipitation.md` — конкретные данные для примеров

## Пример 6: Круговорот воды — ключевые процессы

**Запрос:**
```sparql
#defaultView:Table
SELECT ?process ?processLabel ?description WHERE {
  VALUES ?qid { wd:Q102798 wd:Q179819 wd:Q188488 }
  ?process wdt:P31 ?qid .
  OPTIONAL { ?process schema:description ?description FILTER(lang(?description) = "ru") }
  SERVICE wikibase:label { bd:serviceParam wikibase:language "ru,en". }
}
```

**Что ищет:**
Главные процессы **круговорота воды** в природе. Вода постоянно путешествует: испаряется с океанов, собирается в облака и возвращается дождём обратно.

**Что вернул:**

| Процесс | Описание |
|:---|:---|
| Круговорот воды | Непрерывное движение воды между океаном, атмосферой и сушей |
| Испарение | Переход воды из жидкого состояния в пар |
| Конденсация | Превращение пара в капельки воды (облака) |

**Где пригодится:**
- `water_cycle.md` — основа всей статьи
- `clouds.md`, `precipitation.md` — связь процессов
- `hydrosphere.md`, `atmosphere.md` — междисциплинарные связи

## Пример 7: Биоразнообразие по природным зонам

**Запрос:**
```sparql
#defaultView:Table
SELECT ?zone ?zoneLabel (COUNT(?species) AS ?speciesCount) WHERE {
  ?zone wdt:P31 wd:Q1019915;
        ^wdt:P1435 ?species.
  SERVICE wikibase:label { bd:serviceParam wikibase:language "ru,en". }
}
GROUP BY ?zone ?zoneLabel
ORDER BY DESC(?speciesCount)
LIMIT 10
```

**Что ищет:**
Сколько видов живых существ обитает в разных природных зонах. Это называется **биоразнообразием**. Чем ближе к экватору, тем оно выше.

**Что вернул:**

| Зона | Биоразнообразие |
|:---|:---|
| Тропический лес | Более 50% всех видов Земли |
| Коралловый риф | Около 25% морских видов |
| Умеренный лес | Примерно 10 000 видов растений |
| Тундра | Примерно 1 700 видов растений |

**Где пригодится:**
- `ecosystem.md` — примеры биоразнообразия
- `forest.md`, `desert.md`, `tundra.md` — сравнение зон
- `biosphere.md` — роль биосферы в поддержании жизни

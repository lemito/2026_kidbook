# 12. Строки и текстовая информация: Собираем бусы из букв

![Картинка строки](covers/cover_12_strings.png)

Привет, мастер кода! Мы уже умеем работать с числами, но компьютер — это не только калькулятор. Он умеет читать книги, отправлять сообщения в чатах и даже писать стихи. 

Для всего этого нам нужны **строки**. Давай узнаем, как C++ работает с текстом!

---

## 1. Что такое строка?

Представь себе обычные бусы. Каждая бусинка — это одна буква или символ (в C++ это тип `char`). Когда мы нанизываем эти бусинки на нитку в определенном порядке, у нас получается **строка** (`std::string`).

В строке может быть:
*   Слово: `"Привет"`
*   Целое предложение: `"Я учусь программировать на C++!"`
*   Цифры (но как текст): `"12345"`
*   И даже просто пробел: `" "`

---

## 2. Подключаем «Коробку с буквами»

Чтобы C++ научился понимать длинные строки, нам нужно в самом верху программы добавить специальную библиотеку. Это как достать из шкафа коробку с инструментами для работы с текстом.

```cpp
#include <iostream>
#include <string> // Вот она, наша коробка!
using namespace std;
```

---

## 3. Создаем свою первую строку

Создать строку очень просто. Пишем тип данных `string`, придумываем имя и в двойных кавычках пишем текст:

```cpp
string magicWord = "Абракадабра";
cout << magicWord;
```

**Важно:** Текст всегда должен быть в **двойных кавычках** `" "`. Одинарные кавычки `' '` используются только для одной маленькой бусинки-буквы (`char`).

---

## 4. Как склеивать строки (Конкатенация)

Представь, что у тебя есть две ленты с текстом, и ты хочешь их соединить. В C++ это делается с помощью обычного знака «плюс» `+`. Это называется магическим словом **конкатенация**.

```cpp
string firstName = "Иван";
string lastName = "Иванов";

// Склеиваем имя, пробел и фамилию
string fullName = firstName + " " + lastName;

cout << fullName; // Выведет: Иван Иванов
```

Ты можешь «плюсовать» строки сколько угодно раз, собирая огромные предложения!

---

## 5. Считаем бусинки (Длина строки)

Иногда нам нужно знать, сколько букв в слове. Для этого у строки есть встроенная «линейка» — команда `.length()` или `.size()`.

```cpp
string word = "Компьютер";
cout << word.length(); // Выведет: 9
```

Это очень полезно! Например, если ты делаешь игру, где нужно вводить пароль не короче 8 символов.

---

## 6. Достаем букву по номеру (Индексы)

Каждая буква в строке имеет свой номер, но есть один секрет: **программисты всегда начинают считать с нуля!**

Представь очередь в магазин:
1.  Первый в очереди — это номер **0**.
2.  Второй — номер **1**.
3.  Третий — номер **2**.

Давай достанем буквы из слова `"Кот"`:
```cpp
string pet = "Кот";
cout << pet[0]; // Выведет: К
cout << pet[1]; // Выведет: о
cout << pet[2]; // Выведет: т
```

Если ты попробуешь достать букву номер 10 из слова "Кот", программа расстроится и выдаст ошибку, потому что такой буквы нет!

---

## 7. Ввод строки: Секрет пробела

Ты уже знаешь про `cin >>`. Но со строками он ведет себя как жадина. Если ты введешь `"Привет, мир!"`, то `cin` заберет только первое слово `"Привет"`, а остальное «выбросит».

Чтобы прочитать целую строчку с пробелами, используй волшебную команду `getline`:

```cpp
string message;
cout << "Напиши секретное послание: ";
getline(cin, message); // Читает всю строку целиком до нажатия Enter
cout << "Твое сообщение: " << message;
```

---

## 8. Сравнение строк

Строки можно сравнивать, как числа. Компьютер проверяет их по алфавиту (как в словаре).

```cpp
string a = "Яблоко";
string b = "Яблоко";

if (a == b) {
    cout << "Строки одинаковые!";
}

string word1 = "Абрикос";
string word2 = "Банан";

if (word1 < word2) {
    cout << "Абрикос идет раньше Банана в словаре";
}
```

---

## 9. Полезные трюки

У строк есть много крутых команд. Вот самые популярные:

1.  **`.empty()`** — проверяет, не пустая ли строка. Возвращает `true` (правда), если в строке ничего нет.
2.  **`.clear()`** — мгновенно удаляет всё из строки, делая её пустой.
3.  **`.append()`** — еще один способ добавить текст в конец (похоже на `+`).
4.  **`.substr(откуда, сколько)`** — отрезает кусочек от строки.

Пример с обрезанием:
```cpp
string longWord = "Телевизор";
string smallWord = longWord.substr(0, 4); 
cout << smallWord; // Выведет: Теле
```

## 10. Задание для юного хакера

Попробуй написать программу, которая:
1. Спрашивает имя пользователя.
2. Спрашивает его любимый цвет.
3. Склеивает это в фразу: "Привет, [Имя]! Твой любимый цвет — [Цвет]".
4. Выводит, сколько букв в этой фразе.

---

## Важные советы напоследок:

*   **Не путай `'A'` и `"A"`!** В одинарных кавычках — один символ, в двойных — строка (даже если в ней одна буква).
*   **Пробелы — это тоже символы!** Они занимают место и имеют свой номер (индекс).
*   **Будь внимателен с `getline`!** Если перед ним ты использовал обычный `cin >>`, иногда нужно «очистить» ввод, но об этом мы поговорим в следующий раз.

---

## Итог урока

1.  Строка (`std::string`) — это цепочка символов.
2.  Всегда пиши `#include <string>`.
3.  Считаем символы с **0**, а не с 1.
4.  Склеиваем строки знаком **+**.
5.  Для ввода фразы с пробелами используем **`getline(cin, s)`**.

Теперь ты умеешь общаться с пользователем не только цифрами, но и словами! Ты на шаг ближе к созданию своего собственного текстового квеста или чат-бота. 

Удачи, и пусть твой код всегда компилируется с первого раза!
